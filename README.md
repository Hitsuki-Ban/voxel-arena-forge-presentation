# VoxelArenaForge プレゼン資料(公開用)

VoxelArenaForge(軽量 LLM によるゲームコンテンツ量産の実証プロジェクト)の
発表スライドと配布記事だけを切り出した公開用リポジトリです。

- 入り口: `index.html`(GitHub Pages)
- スライド: `concept-slides.html`
- 配布記事: `articles/`(全6本+付録)

## 更新のしかた

元リポジトリ(private 側)の `docs/presentation/` が原本です。更新時はそちらから
`concept-slides.html`・`articles/`・`assets/` をコピーし直してください。
付録記事の設計ボード画像だけは `assets/design-boards/` に置き、
`articles/appendix-design-boards.html` の参照を `../assets/design-boards/` に
書き換えています(原本は `docs/internal/assets/` を参照)。

実演環境・運用手順(ランブック)は公開対象外です。
